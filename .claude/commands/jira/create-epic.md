# Create Jira Epic

@jira-ticket-manager

## Task
Create a new Jira epic with the following:

**Epic Name**: $ARGUMENTS

## Requirements
1. Create the epic with a comprehensive description using ADF formatting
2. Include sections for:
   - **Overview**: What this epic achieves
   - **Goals**: Clear objectives
   - **Scope**: What's included and excluded
   - **Success Criteria**: How we measure completion
3. Use emoji prefixes (📋 📊 🎯) for visual organization
4. Apply relevant labels for categorization
5. If requirements are provided, break them down into child tickets and link them
6. Confirm successful creation with epic key and link

## Output Format
After creation, provide:
- Epic key (e.g., PROJ-123)
- Direct link to the epic
- Summary of structure created
- List of any child tickets created
