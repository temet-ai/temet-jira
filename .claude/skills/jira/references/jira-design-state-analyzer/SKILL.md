# Jira State Analysis Design

Specialized skill within the `/jira` uber-skill for designing state analysis systems.

## Quick Start

Design a state analysis system:
```
Define state transitions: To Do → In Progress → Review → Done
Track business hours in each state
Calculate bottleneck metrics
```

## What This Skill Does

- **State Design** — Define workflow states
- **Metrics Definition** — Business hours, cycle time
- **Analysis Framework** — Bottleneck detection
- **Reporting** — Summary statistics
- **Optimization** — Workflow improvements
